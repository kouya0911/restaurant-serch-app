レストラン検索アプリ

Udemyの講座で学んだ基礎をベースに、自分なりの実用的なカスタマイズを加えて開発したレストラン検索アプリケーションです。


★アプリの概要

「今すぐ近くの店を探したい」というシーンに特化した検索ツールです。Google Places APIを活用し、リアルタイムでジャンル別・場所別の検索が可能です。


★こだわりのポイント

・実用的なユーザー動線:
詳細ページの実装をあえて行わない代わりに、「店名をワンクリックでコピー ➡ そのまま食べログやGoogleで検索」という、APIの制限を考慮しつつユーザーの利便性を最大化する設計にしました。

・APIコストと速度の最適化:
Google Maps APIの無駄な消費を抑えるため、Next.jsの機能を活用したキャッシュ機構を実装し、パフォーマンスを向上させています。

・セキュアな設計:
SupabaseのRLS（Row Level Security）を使用し、ユーザー情報の安全な保存・管理を行っています。

★ユーザー認証とデータセキュリティ

・安心のログイン機能: Supabase Authを導入し、メールアドレスによるユーザー認証を実装しています。

・マイページ機能: ログインユーザーごとに「保存した住所」や「お気に入り設定」が保持され、デバイスをまたいでも自分の設定が反映されます。

・高度なアクセス制限: 行単位セキュリティ（RLS）により、自分のデータは自分しか閲覧・編集できない堅牢なセキュリティを構築しています。


★使用技術

Frontend: React, Next.js, Tailwind CSS

Backend: Supabase (Auth, RLS)

API: Google Maps Platform (Places, Maps, Geocoding)

Deployment: Vercel
